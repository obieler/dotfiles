dotfiles
========

Config files
------------

* Git - [.gitconfig](.gitconfig)
  * copy to ~
  * update your name and email
* Shutter - [.shutter](.shutter/)
  * copy to ~
* Powerlevel9k - [.powerlevel9k](.powerlevel9k) 
  * copy to ~
  * source this file in .zshrc

---------------------

`./bin/install-atom-packages.sh`
